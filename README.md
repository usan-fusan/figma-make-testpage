# 404 Page Not Found

A clean, modern 404 error page built with React and Tailwind CSS.

## Features

- Centered layout with warm cream background
- Custom document icon with alert symbol
- Responsive design
- Hover effects on the "Go Home" button

## Tech Stack

- React 18.3.1
- Tailwind CSS 4.1.12
- Vite 6.3.5

## Getting Started

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev
```
